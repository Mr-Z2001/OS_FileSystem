## format of userdata.csv
```
each line:
<username>,<password_hashedby_sha224>
```

## format of dir.csv
```
each line:
<directoryID>,<directoryName>,<fileNum>,<fileID1>,<fileID2>,...
```

## format of graph.csv
```
each line:
<parentDirectory>,<childDirectory>
```

## format of file.csv
```
each line:
<fileID>,<fileName>
```
