create [option] <filename> 
open <filename>
cat [option]... <filename> ...
  option:
  -A, --show-all
      equivalent to -vET

  -b, --number-nonblank
      number nonempty output lines, overrides -n

  -e  equivalent to -vE

  -E, --show-ends
      display $ at end of each line

  -n --numer
      number all output lines

  -s, --squeeze-blank
      supperess repeated empty output lines

  -t  equivalent to -vT

  -T, --show-tabs
      display TAB characters as ^T

  -u  (ignored)

  -v, --show-nonpriting
      use ^ and M- notation, except for LFD and TAB

write [option] <filename> [text]

  -a, --append
      append text to the end of file

  -o  overwrite the text.

close <filename>
delete <filename>...
